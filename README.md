# Left Click Vacation

Take a break from holding left click.

This mod lets you toggle automatic block breaking while mining, digging, or dismantling builds. Press your configured key, aim at blocks as usual, and the mod keeps mining without making you hold the mouse button the whole time.

Long mining sessions can be rough on your hand for no interesting gameplay reason. The mod handles the repetitive button holding while still leaving you in control of movement, aiming, tool choice, and combat.

This is not a general autoclicker. It is focused on block breaking and is designed to avoid accidental combat actions.

## What It Does

- Toggles automatic block breaking with a keybind.
- Works with tools like pickaxes, axes, shovels, and other held items.
- Pauses when you switch away from the item you enabled it with.
- Resumes if you switch back before the configurable grace period expires.
- Lets normal durability loss happen without stopping the session.
- Stops if the activation item is dropped, broken, lost, or clearly replaced.
- Does nothing when you are aiming at entities, air, fluids, or unbreakable blocks.
- Shows simple HUD messages when auto click is enabled, paused, or disabled.

## Configuration

You can configure:

- The toggle key.
- The pause grace period.
- Whether enabled, paused, and disabled HUD messages are shown.
- How long fixed HUD messages stay on screen.

The toggle key is unbound by default, so you choose when the feature becomes available.

## Multiplayer

The core feature is client-side and does not require the server to install the mod.

Some servers may consider automatic block breaking to be cheating or otherwise against their rules. Check the rules of the server you play on.
